This library helps to deal with Hibernate entities and collections in GWT applications - using the simplest available solutions.
You can use this library as a [Gilead](http://noon.gilead.free.fr/gilead/) alternative for simple projects with Hibernate persistence (please note that Gilead currently has a wider set of features).

For information:
  * see the [WIKI](Introduction.md)
  * the [generated Maven site](http://www.erinors.com/static/developer/project/hibernate-pojo-bridge/)
  * [API Javadocs](http://www.erinors.com/static/developer/project/hibernate-pojo-bridge/apidocs/index.html)
  * [Unit test coverage](http://www.erinors.com/static/developer/project/hibernate-pojo-bridge/cobertura/index.html) is getting better and better :)

A work-in-progress demo application is available in SVN: [hpb-example-webapp](http://hibernate-pojo-bridge.googlecode.com/svn/trunk/hpb-example-webapp/)

Compatibility:
**[Branch 2.0.0](http://code.google.com/p/hibernate-pojo-bridge/source/browse/#svn/branches/2.0.0/hibernate-pojo-bridge) is compatible with Hibernate 3.5** [Branch 1.0.9](http://code.google.com/p/hibernate-pojo-bridge/source/browse/#svn/branches/1.0.9/hibernate-pojo-bridge) is compatible with Hibernate 3.3
